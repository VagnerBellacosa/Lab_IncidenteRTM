Interactive **mainframe architecture simulators** designed to help students and professionals understand the internal structure of IBM mainframe systems.

This project provides **visual and interactive demonstrations** of core mainframe concepts inspired by the **IBM Systems Service Structure course**.

---

# 🎯 Project Objective

Learning mainframe internals can be difficult because most concepts are abstract and hidden inside the operating system.

This project provides **interactive simulations that demonstrate how internal components behave**.

Instead of only reading theory, users can explore:

✔ System initialization  
✔ Task dispatching  
✔ I/O processing  
✔ Memory addressing  
✔ Storage management  
✔ Error recovery  
✔ IPL troubleshooting  

All simulations run directly in the browser.

# 🧠 Included Simulators

| Unit   | Topic                                |
| ------ | ------------------------------------ |
| Unit 2 | Operating Environment Initialization |
| Unit 3 | Task Management                      |
| Unit 4 | Input/Output Supervisor              |
| Unit 5 | Addressability                       |
| Unit 6 | Storage Management                   |
| Unit 7 | Recovery Termination Manager         |
| Extra  | IPL Troubleshooting Lab              |

Each simulator reproduces real behaviors found in systems like **z/OS**.

Examples include:

- system IPL sequence
- task dispatching
- memory frame allocation
- I/O channel programs
- ABEND recovery
- IPL diagnostics
