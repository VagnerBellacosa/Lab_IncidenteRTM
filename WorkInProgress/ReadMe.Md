# 📚 Overview

This repository provides a collection of **interactive simulators designed to explain internal architecture concepts of IBM mainframe systems**.

The simulators are inspired by the **IBM Systems Service Structure course**, transforming theoretical concepts into **visual and interactive experiences** that run directly in the browser.

Instead of only reading documentation, users can **experiment with how the system behaves internally**.

---

# 🧠 Mainframe Virtual Lab

The repository simulates internal components of systems such as **z/OS**, including:

| Module | Topic                                |
| ------ | ------------------------------------ |
| Unit 2 | Operating Environment Initialization |
| Unit 3 | Task Management                      |
| Unit 4 | Input/Output Supervisor              |
| Unit 5 | Addressability                       |
| Unit 6 | Storage Management                   |
| Unit 7 | Recovery Termination Manager         |
| Extra  | IPL Troubleshooting Lab              |

Each module demonstrates how real components behave inside a mainframe system.

Examples include:

- IPL startup sequence
- task dispatching
- I/O operations
- virtual memory management
- address translation
- system recovery
- ABEND handling
